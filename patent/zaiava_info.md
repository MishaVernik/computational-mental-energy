## Дані для заповнення

### Назва винаходу

Система та спосіб обробки багатоканальних EEG-даних для обчислення показника обчислювальної ментальної енергії (CME) із застосуванням квантового обчислювального модуля та метаевристичної оптимізації.

### Рекомендована класифікація за МПК (IPC)

- **G06N 10/00** – Квантові обчислення, тобто обробка інформації на основі квантово-механічних явищ
- **G06N 3/08** – Навчальні методи (нейронні мережі)
- **A61B 5/0476** – Електроенцефалографія
- **A61B 5/00** – Вимірювання для діагностичних цілей; ідентифікація осіб
- **G16H 50/20** – ІКТ, спеціально адаптовані для медичної діагностики, медичного моделювання або медичного аналізу даних; ІКТ для виявлення, моніторингу або моделювання епідемій або пандемій – із використанням методів машинного навчання

## Порядок подання (КПІ)

Згідно інформації відділу трансферу технологій КПІ ([science.kpi.ua/invent/](https://science.kpi.ua/invent/)):

1. **Заповнити онлайн-форму** "Повідомлення про створення ОПІВ" (об'єкта права інтелектуальної власності).

2. **Пройти оцінку доцільності** набуття прав на службові ОПІВ згідно методики оцінювання і рейтингування.

3. **Підготувати документи заявки** українською мовою:
   - заява про видачу патенту;
   - опис винаходу;
   - формула винаходу;
   - креслення;
   - реферат.

4. **Подати заявку**:
   - особисто: кімн. 1-3.2 (1 поверх бібліотеки КПІ ім. Ігоря Сікорського);
   - або онлайн: надіслати матеріали заявки на ttransferkpi@gmail.com.

## Підготовка креслень до подання

Фігури у цьому пакеті підготовлені як Mermaid-діаграми у файлах `kresl_fig1.md` – `kresl_fig4.md`. Для подання заявки їх необхідно:

1. Відкрити кожен файл у редакторі з підтримкою Mermaid (наприклад, VS Code з розширенням Mermaid Preview, або онлайн-редактор https://mermaid.live/).
2. Експортувати кожну діаграму у формат PNG або SVG.
3. При необхідності – перенести на аркуш формату А4 з номером заявки у верхньому правому куті.
4. Нумерація: Фіг. 1, Фіг. 2, Фіг. 3, Фіг. 4.

## Примітки

- Заявка на винахід повинна стосуватися одного винаходу або групи винаходів, пов'язаних єдиним винахідницьким задумом (вимога єдності винаходу). У цьому випадку – система та спосіб пов'язані єдиним задумом обчислення CME.
- Усі документи подаються українською мовою.
- Перед фінальним поданням рекомендується узгодити текст формули із патентним повіреним або спеціалістом відділу трансферу технологій КПІ.
